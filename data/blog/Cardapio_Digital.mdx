## Introdução

O **Java Cardápio Digital** é uma aplicação full stack desenvolvida para modernizar a gestão e apresentação de cardápios em restaurantes, lanchonetes e estabelecimentos do ramo gastronômico. Com backend robusto em Java Spring Boot e frontend responsivo em React com TypeScript, o projeto demonstra integração eficiente entre tecnologias modernas e proporciona uma experiência de usuário fluida e profissional.

## Etapas do Projeto

### Backend — Java Spring Boot

- Projeto iniciado com Spring Boot, garantindo agilidade e segurança no backend.
- Configuração de banco de dados e criação dos modelos de domínio.
- Desenvolvimento de controladores RESTful para operações CRUD.
- Utilização do Spring Data JPA para simplificação do acesso e manipulação de dados.

### Frontend — React e TypeScript

- Implementação da interface com React, garantindo interatividade e responsividade.
- Uso de TypeScript para melhor manutenção e organização do código.
- Criação de componentes reutilizáveis e utilização de React Hooks para gerenciar estados e efeitos.
- Conexão com backend via Axios e gerenciamento do estado do servidor com React Query, otimizando buscas, cache e sincronização dos dados.

## Tecnologias Utilizadas

- **Java**
- **Spring Boot**
- **React**
- **TypeScript**
