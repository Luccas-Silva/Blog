## Introdução

O **Fiap Challenge Salesforce** propõe uma experiência digital inovadora para os usuários da Salesforce, focando em acessibilidade, usabilidade e integração. O projeto foi desenvolvido a partir de feedbacks reais dos próprios usuários da plataforma, e busca criar um site mais inclusivo, explicativo e agradável para todos os públicos.

<TOCInline toc={props.toc} exclude="Introdução" />

## Objetivos do Projeto

- Impressionar a Salesforce com um site centrado no usuário, com acessibilidade aprimorada.
- Implementar recursos visuais otimizados, navegação simplificada e opções de iluminação.
- Criar páginas explicativas para detalhar serviços, tirar dúvidas e evidenciar funcionalidades.
- Desenvolver um ChatBot interativo para auxiliar navegação e fornecer contato direto com a Salesforce.

## Funcionalidades de Acessibilidade

- Otimização visual para diferentes perfis de usuários.
- Navegação intuitiva e simplificada.
- Alternância de temas/iluminação.
- ChatBot acessível para suporte e localização rápida no site.

## Backend Java: Robustez e Arquitetura em Camadas

A aplicação Java garante a integração eficiente entre o front-end e o banco de dados, suportando operações simultâneas com consistência e confiabilidade. Destaques:

- **Criação e Gerenciamento de Tabelas**: Estrutura o banco de dados conforme as demandas da aplicação.
- **APIs RESTful**: Facilita a comunicação entre front-end e back-end.
- **Arquitetura em Camadas**:
  - **Resource**: Define os endpoints da API.
  - **Service**: Implementa as regras de negócio.
  - **Model**: Estrutura das entidades.
  - **DAO**: Organiza o acesso aos dados, deixando o sistema mais flexível e testável.

## Benefícios

- Arquitetura robusta e escalável, pronta para grandes volumes de dados.
- Produtividade elevada por meio de APIs e abstração de dados.
- Consistência e confiabilidade garantidas através do gerenciamento centralizado de dados.

## Tecnologias Utilizadas

- **Frontend**: HTML, CSS, React, TypeScript
- **Backend**: Java, Apache Maven
- **Design**: Figma
