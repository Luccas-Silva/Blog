## Introdução

O **Chess System Java** é um projeto que traz o clássico jogo de xadrez para o terminal, totalmente implementado em Java. Com arquitetura orientada a objetos e lógica fiel às regras oficiais do xadrez, o sistema oferece uma experiência completa e educativa tanto para jogadores quanto para estudantes de programação.

<TOCInline toc={props.toc} exclude="Introdução" />

## Ideia do Projeto

O objetivo central do projeto é desenvolver um jogo de xadrez plenamente funcional, aproveitando os conceitos de orientação a objetos em Java. O sistema gerencia tabuleiro, peças, jogadas válidas, regras especiais e apresenta todas as informações via interface de console.

## Como Iniciar o Jogo

1. Compile o projeto conforme instruções do repositório.
2. Entre na pasta `bin` do projeto.
3. Inicie o Git Bash e execute o comando:
   ```bash
   java application/Program
   ```
4. O jogo será iniciado no terminal, pronto para jogar!
