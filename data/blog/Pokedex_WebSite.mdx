## Introdução

O **Pokédex WebSite** é uma plataforma web desenvolvida em Angular que traz para o navegador uma experiência completa de consulta à Pokédex. Utilizando a [PokéAPI](https://pokeapi.co/), o projeto entrega uma lista detalhada de Pokémon, cada um com seu próprio cartão informativo, imagens e dados diretamente da API oficial – tudo com um visual moderno e responsivo.

## Ideia do Projeto

A proposta central foi criar uma ferramenta prática para fãs e desenvolvedores explorarem o universo Pokémon de maneira simples e visual. O app exibe uma lista abrangente de espécies, permitindo ao usuário navegar, pesquisar e visualizar detalhes dos Pokémon com facilidade.

## Tecnologias Utilizadas

- **HTML**: Estrutura da aplicação.
- **SCSS**: Estilização moderna e componentes visuais elegantes.
- **TypeScript**: Tipagem forte e manutenção facilitada.
- **Angular**: Framework robusto para construir SPAs interativas e rápidas.

## Como Rodar o Projeto

1. Instale as dependências com o Angular CLI.
2. Execute o comando:
   ```bash
   ng serve
   ```
3. Acesse [http://localhost:4200/](http://localhost:4200/) para visualizar a aplicação. O app recarrega automaticamente a cada alteração.
