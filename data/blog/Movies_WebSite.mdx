## Introdução

O **Movies WebSite** é uma aplicação web desenvolvida para quem adora cinema e tecnologia! O projeto oferece um catálogo completo de filmes, onde usuários podem explorar títulos, visualizar informações detalhadas e deixar suas avaliações. Combinando um frontend em React e um backend robusto em Java com Spring Boot, o sistema entrega performance, usabilidade e visual moderno.

<TOCInline toc={props.toc} exclude="Introdução" />

## Objetivo do Projeto

A proposta é criar uma plataforma dinâmica de filmes, permitindo que usuários:

- Naveguem por um catálogo atualizado de títulos.
- Consultem detalhes de cada filme.
- Avaliem e compartilhem opiniões.
- Experienciem uma interface responsiva e intuitiva.

## Tecnologias Utilizadas

- **Frontend:** React, HTML, CSS
- **Backend:** Java, Spring Boot
- **Banco de Dados:** Persistência das informações dos filmes e avaliações

O Movies WebSite é a demonstração prática de como unir as melhores tecnologias de frontend e backend para criar uma solução completa, moderna e envolvente.
